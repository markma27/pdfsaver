# Test PDF Files

This folder is for storing test PDF files.

## Notes

- **Do NOT commit actual PDF files to Git repository**
- For local development and testing only
- It is recommended to exclude PDF files in this folder using `.gitignore`

## Test File Types

It is recommended to prepare the following types of test PDFs:

1. **dividend-statement.pdf** - Dividend statement (text PDF)
2. **buy-contract.pdf** - Buy contract (text PDF)
3. **sell-contract.pdf** - Sell contract (text PDF)
4. **holding-statement.pdf** - Holding statement (text PDF)
5. **tax-statement.pdf** - Tax statement (text PDF)
6. **scanned-document.pdf** - Scanned PDF (no text layer, requires OCR)

## Usage

Place test PDF files in this folder, then upload them in the application for testing.
